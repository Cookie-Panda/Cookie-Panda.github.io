**Wanxian - Biome Generator for Minecraft Bedrock Edition**
Wanxian is a webapp designed to create custom Minecraft Bedrock Edition biome Addons.
üöÄ Features:
 * Comprehensive Biome Configuration: Easily define climate settings, surface builders, mountain parameters, and more.
 * Dual Biome Support: Create and manage both Custom Biomes (Behavior Pack) and Client Biomes (Resource Pack).
 * Dynamic Fog Creator: A dedicated editor to build, import, and export custom fog settings with support for different environments like air, water, and lava.
 * Pack Generation: Export your creations directly as .mcaddon files containing both your Behavior Pack and Resource Pack.
 * JSON Management: Import existing biome files to edit them or save your configurations as presets in local storage.

 * Climate: Temperature, downfall, and snow accumulation.
 * Surface Builder: Support for Overworld, Frozen Ocean, Mesa, Swamp, and Capped builders.
 * Visuals: Map tints, sky color, water appearance, foliage, and grass appearance.
 * Environmental: Precipitation modes, ambient sounds, and biome music.
 * Generation Rules: Replacement rules with target dimension selection for Overworld or Nether(Note: Nether Replacement Current Works in latest Beta/Preview Versions).
 * Beta/Preview: It is currently supporting latest Beta/Preview Version.
üíª Interface:
 * HTML5/CSS3: Expressive UI using Material 3 design principles.
 * Vanilla JavaScript: Core logic for JSON generation and form management.
 * JSZip: Library used for generating and packaging .mcpack and .mcaddon files on the fly.
 * The App is Friendly for Portrait Primary Device i.e. Mobile devices.
 * This App is PWA so so can be installed and runs offline.
üìñ How to Use:
 * Define Core Info: Enter your Format Version and Biome Identifier (e.g., namespace:biome_name).
 * Toggle Components: Enable the components you need and fill in the parameters.
 * Add Decorations: Use the "Decorative Presets" section to include pre-defined features and their associated tags.
 * Create Custom Fog: Use the Fog Creator to define unique environmental fog.
 * Generate & Export: Click the robot icon to generate the JSON and download your files or the complete addon.
